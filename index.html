Python Project: IMDB Data Set Analysis

Dataset & prep: IMDB movies and reviews. Standardized columns to lowercase; removed rows with critical missing values; dropped non-analytic fields (e.g., long text/visual fields) to reduce noise.

  • Exploratory analysis of ratings, duration, genres, franchises, and review features.
  • Finding 1: Genre alone does not explain high ratings; focus on script quality, direction, and acting instead.
  • Finding 2: Viewer satisfaction is highest for runtimes roughly 90–180 minutes.
  • Finding 3: Franchises have higher average ratings, but sequels often decline—two installments is typically the sweet spot.
  • Managerial guidance: invest in core quality drivers; tailor release strategy to runtime (streaming vs. theatrical); avoid unnecessary sequels without clear demand.
